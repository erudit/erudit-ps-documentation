<state>
=========

+-------------------------+-----------------+
| Appears in              | Frequency       |
+=========================+=================+
| :ref:`element-aff`      | Zero or once    |
+-------------------------+-----------------+
| :ref:`element-corresp`  | Zero or once    |
+-------------------------+-----------------+

Identifies the state in an address. It is not a mandatory element in affiliation.

Example:

.. code-block:: xml

   ...
   <aff id="aff01">
     ...
     <state>Quebec</state>
      ...
   </aff>
   ...
