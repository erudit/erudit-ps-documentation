.. _element-copyright-year:

<copyright-year>
================

+-----------------------------+--------------------+
| Appears in                  | Frequency          |
+=============================+====================+
| :ref:`elemento-permissions` | Zero or more times |
+-----------------------------+--------------------+

Identifies the year of copyright as four digits without spaces.

Example:

.. code-block:: xml

    ...
     <article-meta>
       ...
        <permisssions>
           ...
            <copyright-year>2015</copyright-holder>
           ...
        </permissions>
        ...
     </article-meta>
