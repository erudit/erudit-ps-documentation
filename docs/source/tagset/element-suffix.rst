.. _element-suffix:

<suffix>
========

+----------------------+-----------------+
| Appears in           | Frequency       |
+======================+=================+
| :ref:`element-name`  | Zero or once    |
+----------------------+-----------------+

Specifies the suffixes of the author name such as “Sr.”, “Jr.”, “II”, etc.

Exemplo:

.. code-block:: xml

    ...
    <name>
        <surname>Santos</surname>
        <given-names>João da Silva</given-names>
        <suffix>Jr</suffix>
        ...
    </name>
    ...
